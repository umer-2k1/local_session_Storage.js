Using Local Storage Api

Local Storage let you save data in your browser for the longer run. The data is stored against the website address in the form of key/value pair.

How store data in LS

The data is only stored as String, first we need to convert Array or Object into String using JSON.stringify

To see data in Browser

  1. Go to Developer Tool by pressing F12
  2. Press Application tab
  3. In the local Storage section expand Local Storage
